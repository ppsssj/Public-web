# AICS Public Web

간단한 소개

- 이 리포지터리는 AICS(연구소 또는 조직) 퍼블릭 웹사이트 프론트엔드 소스입니다. React 기반으로 작성되어 있으며, 단일 페이지(홈/섹션) 형태로 구성됩니다.

요구사항(Prerequisites)

- Node.js 18 이상 권장
- npm 또는 yarn

빠른 시작

1. 의존성 설치

```bash
npm install
```

2. 로컬 개발 서버 실행

```bash
npm run dev
```

브라우저에서 http://localhost:5173 또는 터미널에 표시되는 주소를 엽니다.

프로덕션 빌드

```bash
npm run build
```

프로젝트 구조(주요 파일)

- `src/` : 애플리케이션 소스
  - `App.jsx` : 페이지 레이아웃 및 섹션 정의
  - `main.jsx` : 엔트리 포인트
  - `styles.css` : 전역 스타일
  - `components/` : 재사용 가능한 컴포넌트 모음 (예: `SectionNav.jsx`, `Header.jsx`, `Footer.jsx`)
  - `data/` : 페이지 컨텐츠 및 네비게이션 데이터

테스트 및 확인 절차

1. 개발 서버 실행 후 페이지에서 하단 네비게이션 확인
2. 버튼 클릭 시 스무스 스크롤로 해당 섹션으로 이동하는지 확인
3. 스크롤로 섹션이 중앙 근처에 진입하면 해당 버튼의 테두리가 `#5f788b`로 바뀌는지 확인
